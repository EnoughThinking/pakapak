Pakapak
=========

npm + bower + other conf = pakapak

## Installation

  npm install pakapak --save-dev

## Usage

  var pakapak = require('pakapak');

  var result = collect(["project/**/pakapak.json"], true);

  console.log(' { a: 1, app: 1, work: 1 }', result);

## Tests

  npm test
